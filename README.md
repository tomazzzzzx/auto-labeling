# Auto Labeling

Automated data labeling pipeline using LLMs and active learning.

## Pipeline
1. LLM-based initial labeling with confidence scores
2. Active learning selects uncertain samples for human review
3. Iterative refinement with human-in-the-loop
4. Quality metrics: inter-annotator agreement, label distribution

## Supported Tasks
- Text classification
- Named entity recognition
- Sentiment analysis
- Image captioning
